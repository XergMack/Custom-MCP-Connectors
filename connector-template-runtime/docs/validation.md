# Validation Standard

A connector is not valid because it builds.

A connector is valid when it proves:

1. initialize works
2. tools/list works
3. at least one real tools/call works
4. the live endpoint is attachable in ChatGPT
5. evidence is stored in the repo

## Recommended Minimum Evidence

- initialize-response.json
- tools-list-response.json
- one real safe tool call result
- working-endpoint.txt

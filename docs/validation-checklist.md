# Validation Checklist

1. Confirm raw vendor API call succeeds
2. Confirm auth token/key is valid
3. Confirm base URI is correct
4. Confirm direct core function succeeds
5. Confirm local /healthz
6. Confirm local read route
7. Confirm local write route
8. Confirm Azure image build
9. Confirm Azure /healthz paths
10. Confirm Azure read route
11. Confirm Azure write route
12. Rotate secrets after proof

# ServiceDesk MCP API Map

Purpose:
Capture the full ServiceDesk Plus API shape for the rebuilt thin MCP connector.

How to use:
For each family, document:
- endpoint(s)
- HTTP method(s)
- read/write
- query parameter shape
- input_data shape
- required fields
- response shape
- known working example
- notes / quirks

Rule:
Do not guess the API shape.
Only record shapes that are proven by docs or live testing.
